## 코드

### 1. clock()을 이용한 실행 시간 측정

``` c
#include <stdio.h>
#include <time.h>

int main() {
	clock_t start, stop;
	double duration;
	long long sum = 0;
	start = clock();
	for (long long i = 0; i < 1000000000; i++) {
		sum += i;
	}
	stop = clock();
	duration = (double)(stop - start) / CLOCKS_PER_SEC;
	printf("sum의 값: %lld\n", sum);
	printf("수행시간은 %lf초입니다.\n",duration);
}

```

만약에 for (long long i = 0; i < 1000000000; i++) ; 이런식으로 쓰게 되면 0.00초로 나오게 되는데, 이는 컴파일러가

프로그램 밖에서 관찰 가능하지 않는 부분은 컴파일 과정에서 삭제해버리기 때문이다.

이를 Dead Code Elimination (죽은 코드 제거)라고 하고 컴파일러는 이 부분을 아예 기계어로 번역하지 않는다.

### 2. time()을 이용한 실행 시간 측정

``` c
#include <stdio.h>
#include <time.h>

int main() {
	time_t start, stop;
	double duration;
	long long sum = 0;
	start = time(NULL);
	for (long long i = 0; i < 1000000000; i++) {
		sum += i;
	}
	stop = time(NULL);
	duration = (double)difftime(stop, start);
	printf("sum의 값: %lld\n", sum);
	printf("수행시간은 %lf초입니다.\n", duration);
}

```
